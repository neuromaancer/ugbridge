# Fonts

`ukij-ekran.woff2` is a web subset of `UKIJ Ekran`, downloaded from the
[UKIJ Uyghur Unicode fonts page](https://ukij.org/fonts/).

The UKIJ page states that its fonts are distributed under LGPL and SIL Open
Font License terms. The web subset keeps only the Latin punctuation, Arabic
script ranges, and bidi controls needed by UG Bridge.
